e-plantShopping

Paradise Nursery - Online Plant Shop

e-plantShopping is a React + Redux based web application for an online plant nursery called **Paradise Nursery**.  
The application allows users to browse plants, add them to a cart, update quantities, and proceed to checkout.

---

Features

- Landing page with background image
- Product listing page with multiple plant categories
- Add to Cart functionality using Redux
- Increment and Decrement product quantity
- Dynamic total cost calculation
- Shopping cart page with checkout option
- Continue shopping functionality

---

Technologies Used

- React.js
- Redux Toolkit
- CSS
- JavaScript (ES6)
